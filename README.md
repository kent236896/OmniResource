# 🚀 OmniResource Downloader

**The Ultimate AI-Powered Media Discovery & Management Engine.**

Stop wasting hours manually searching for the perfect stock assets. **OmniResource Downloader** is a high-performance, intelligent desktop and web application designed to orchestrate your creative workflow by unifying global stock platforms into a single, AI-enhanced interface.

---

## ✨ Why OmniResource?

In a world of fragmented content, OmniResource brings everything together. Whether you are a content creator, designer, or developer, our engine empowers you to find, tag, and manage high-fidelity images and videos with unprecedented speed.

> "Don't just search. Discover with intelligence."

---

## 🌟 Key Features

### 🔍 Multi-Platform Orchestration
Search across **Pexels**, **Pixabay**, and **Coverr** simultaneously. Filter by orientation, quality, and media type to find exactly what you need in seconds.

### 🧠 Intelligent Keyword Expansion
Powered by the **Datamuse API**, our discovery engine expands your simple prompts into a rich web of related keywords, ensuring you never miss the perfect asset due to a limited vocabulary.

### 🏷️ Automatic AI Tagging (CLIP)
Every asset you download is automatically analyzed by the **OpenAI CLIP model** (`Xenova/clip-vit-base-patch32`). Our local AI engine "sees" your images and videos, applying relevant tags like *nature*, *urban*, or *minimalist* without you lifting a finger.

### 📂 Pro-Grade File Management
*   **SQLite-Powered Library:** All metadata and AI tags are stored in a local SQLite database for lightning-fast retrieval.
*   **Tag-Based Filtering:** Search your local library using the tags generated by our AI.
*   **Concurrent Downloads:** High-speed, multi-threaded download management that respects your system's CPU architecture.

### 💻 Desktop & Web Ready
Run it in your browser or launch it as a native **Electron** desktop application. Experience a fluid, high-density UI built with **Tailwind CSS** and **Framer Motion**.

## 🚀 Getting Started

### 1. Prerequisites
*   Node.js (v18 or higher)
*   API Keys for [Pexels](https://www.pexels.com/api/), [Pixabay](https://pixabay.com/api/docs/), and [Coverr](https://coverr.co/api).

### 2. Installation
```bash
# Clone the repository
git clone https://github.com/your-repo/omniresource-downloader.git

# Install dependencies
npm install
```

### 3. Configuration
Add your API keys to the **Secrets** panel within the application settings or set them as environment variables:
*   `PEXELS_API_KEY`
*   `PIXABAY_API_KEY`
*   `COVERR_API_KEY`

### 4. Run the Application
**For Web:**
```bash
npm run dev
```
**For Desktop (Electron):**
```bash
npm run electron:dev
```
